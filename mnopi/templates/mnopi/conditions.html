Política de datos de mnopi

El plugin de mnopi recoge varios datos sobre las páginas que visitas. En cualquier momento puede ser desactivado, de modo que el guardado y envío de datos queda detenido completamente. Aun así, al hacer login es posible elegir qué datos quieres que se recopilen.

Los datos recogidos por defecto son:

  • Direcciones de las páginas visitadas (por ejemplo, www.google.es)
  • Búsquedas en los principales buscadores (Google, Yahoo...)
  • Código HTML de las páginas web visitadas
